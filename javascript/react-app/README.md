# Sapling SDK: Example React App

This app is an example of integrating Sapling into a React application, built with [Vite](https://vite.dev/).
Sapling is declared in `package.json` and installed as a dependency through npm.


## Installation

- `npm install`
- Register for an account at [https://sapling.ai](https://sapling.ai/user/register), then sign in.
- Generate a development API key in your dashboard.
- Replace `<YOUR_API_KEY>` in `src/App.jsx` with your Sapling API key.

## Running

Run `npm run dev`

This runs the app in the development mode.

Then visit [http://localhost:5173](http://localhost:5173) to view the app in your browser.
