# sapling-samples

Examples of how you can use Sapling's JavaScript and Python libraries.

To run the samples, register for an account at [https://sapling.ai](https://sapling.ai/user/register)
and generate a development API key in your dashboard.

| Sample | Description |
|--------|-------------|
| [`html/`](html/) | Minimal HTML page using the Sapling browser SDK |
| [`editors/`](editors/) | Sapling SDK integrated with rich-text editors: CKEditor, Froala, Lexical, Quill, TinyMCE and Tiptap |
| [`javascript/script/`](javascript/script/) | Node.js scripts using `@saplingai/sapling-js` (ES module and CommonJS) |
| [`javascript/react-app/`](javascript/react-app/) | React app built with Vite |
| [`javascript/vue-app/`](javascript/vue-app/) | Vue 3 app built with Vite |
| [`javascript/angular-app/`](javascript/angular-app/) | Angular app |
| [`javascript/express-app/`](javascript/express-app/) | Express backend that proxies Sapling API calls so the API key stays private, with a React frontend |
| [`python/scripts/`](python/scripts/) | Python scripts using the `sapling-py` client |
| [`dart/`](dart/) | Dart script that calls the HTTP API and applies edits to text |
